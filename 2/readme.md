# JS Native Methods & Expressions


- $for$ - $O(n)$

- Array.pop( ) - $O(1)$
- Array.push( ) - $O(1)$
- Array.shift( ) - $O(n)$
- Array.unshift( ) - $O(n)$
- Array.reduce( ) - $O(n)$ if callback is linear if its not + callback's complexity  
- Array.sort() - $n*logn$ (depends, in most cases)
